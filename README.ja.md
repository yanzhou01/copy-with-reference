# Copy with Page Reference

ウェブページのテキストをコピーするときに、URLや引用符を自動で付けてくれる Tampermonkey スクリプトです。

[English](README.md) | [中文](README.zh.md)

---

## インストール方法

1. ブラウザに [Tampermonkey](https://www.tampermonkey.net/) 拡張機能をインストール
2. 拡張機能管理画面を開き、**「ユーザースクリプトの許可」または「開発者モード」を有効化**する（Chrome 系ブラウザで必要な場合あり）
3. `copy-with-reference.user.js` をブラウザで開く
4. Tampermonkey のインストール画面が表示されるので「インストール」をクリック

---

## 使い方

### `Ctrl + C` — URL付き引用コピー

ページ上のテキストを選択して `Ctrl+C` を押すと、ページ URL とコードブロックを付けた形式でコピーされます。

**出力例：**
```
https://example.com/some/page
```

---

### `Alt + C` — 引用ブロックコピー

テキストを選択して `Alt+C` を押すと、各行の先頭に `> ` を付けた Markdown 引用形式でコピーされます。

**出力例：**
```
> 選択したテキストの1行目
> 2行目
> 3行目
```

お客様の問い合わせなど複数行のメッセージを引用するとき、1行ずつ手動で `>` を付ける手間がなくなります。Slack や Notion への引用に便利です。

---

## その他

- テキストを選択していない状態で `Ctrl+C` を押した場合は通常のコピー動作になります
- コピー成功時はプレビューポップアップが表示されます。`Escape` キー、`✕` ボタン、または背景クリックで閉じられます
- すべてのウェブサイトで動作します（`*://*/*`）
- Mac 環境にて [Claude Code](https://claude.ai/code) を使って作りました
